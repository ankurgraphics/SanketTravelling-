## Images to add
Create these files inside `assets/images/` if you want to replace the gradient placeholders:

### Logo and favicon
- `logo.png` — 300×300 px, transparent PNG
- `favicon.png` — 32×32 px or 48×48 px

### Home hero
- `hero-home.webp` — 1600×900 px
- `hero-home-mobile.webp` — 1080×1350 px (optional)

### Page banners
- `banner-packages.webp` — 1600×600 px
- `banner-gallery.webp` — 1600×600 px
- `banner-about.webp` — 1600×600 px
- `banner-contact.webp` — 1600×600 px

### Package cards

## GitHub Pages
After upload:
1. Push all files to your GitHub repository.
2. Go to **Settings → Pages**.
3. Select branch `main`.
4. Set folder to `/root`.
5. Save and wait for deployment.
